## 🛡️ نبذة عن المشروع (About the Project)

**Ventrilo-Ghost** هو إطار عمل (Framework) متطور للأمن السيبراني تم تطويره بواسطة **SayerLinux**. يهدف النظام لتوفير ترسانة متكاملة لعمليات الفحص، الاستغلال، والتحكم عن بعد (C2) بشكل آلي واحترافي.

**Ventrilo-Ghost** is an advanced cybersecurity framework developed by **SayerLinux**. It provides a comprehensive arsenal for automated scanning, exploitation, and Command & Control (C2) operations.

---

## 🔥 الميزات الرئيسية (Key Features)

- **[☢] AUTO-PWN**: نظام هجوم آلي شامل لاكتشاف واستغلال الثغرات.
- **[1] Deep Spider**: زحف عميق في المواقع لاكتشاف الروابط والمسارات المخفية.
- **[2] OSINT**: أدوات جمع معلومات عن النطاقات الفرعية والبنية التحتية.
- **[3] SQLi/LFI Attack**: محرك متخصص في هجمات الحقن وتضمين الملفات.
- **[5] C2 Dashboard**: لوحة تحكم ويب مركزية لإدارة السيرفرات والأجهزة المتصلة.
- **[6] Ghost Encryptor**: أدوات تشفير وفك تشفير البيانات الحساسة.
- **[7] Passive Recon**: استطلاع سلبي لجمع المعلومات دون لفت الانتباه.

## ⚙️ التثبيت (Installation)

### 1. المتطلبات (Prerequisites)
تأكد من تثبيت **Python 3.8+** و **pip3**.

### 2. التثبيت السريع (Quick Install)
```bash
# تحميل المستودع
git clone https://github.com/SaudiLinux/Ventrilo-Ghost-Framework.git
cd Ventrilo-Ghost-Framework

# تثبيت المتطلبات تلقائياً
python3 setup.py
```

أو يدوياً:
```bash
pip install -r requirements.txt
```

## 🚀 التشغيل (Usage)

### تشغيل المحرك الرئيسي (Main Controller)
```bash
python3 main.py
```

### تشغيل لوحة التحكم (C2 Dashboard)
```bash
cd c2_server
python3 app.py
```
ثم توجه إلى: `http://localhost:5000`
